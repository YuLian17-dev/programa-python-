Problema 3 - Reabastecimiento de inventario
Descripción
Este proyecto en Python resuelve un problema de inventario utilizando una matriz con la estructura:

[Código Artículo, Nombre, Stock Actual, Stock Mínimo Requerido]

El programa calcula la cantidad exacta que debe pedirse para cada artículo:

- Si el stock actual es menor al stock mínimo, se solicita la diferencia.
- Si el stock actual es suficiente, la cantidad a pedir es cero.
